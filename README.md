#gitThreeView

gitThreeView is a simple app which you can use to visualize your Github repositories in 3D.

Enjoy.


##Technologies

Ruby on Rails

Javascript

jQuery

Github API / Octokit.rb

Three.js (webGL 3D framework)

Tween.js (animate 3D models)

Underscore.js

Postgresql
